# Koji
Command line BitTorrent client for [nyaa.si](https://nyaa.si/)

### Requirements
* Python
* Pipenv

### Installing
Clone this repo
```sh
git clone https://github.com/kcchik/koji.git

cd koji
```

Install dependencies and start
```sh
pipenv shell

pipenv install

python koji
```
