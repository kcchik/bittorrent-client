# Koji
Command line BitTorrent client for [nyaa.si](https://nyaa.si/)

### Requirements
* Python
* Pipenv

### Installing
Install dependencies
```sh
pipenv shell

pipenv install
```

Start and show help message
```sh
python koji
```
