# Koji
Command line BitTorrent client for [nyaa.si](https://nyaa.si/)

### Requirements
* Python
* Pipenv

### Installing
Install dependencies
```sh
pipenv shell

pipenv install
```

Show usage instructions
```sh
python koji
```
